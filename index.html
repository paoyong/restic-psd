Simple, robust, immutable backup system for your working files. Build on restic.

Simple

Install, setting folders to be watched. In those folders, just add "WORKING" to the filename. Those files will be backed up, every 30 minutes and once daily.

Robust

This program will set up Windows Task Scheduler that runs generated backup script every 30 minutes and daily, and gives you a GUI to run restic restore commands.

On top of that, this is an immutable program. That means, once you set it up, there's no changing things. It's a fixed size wrench. This is so there is as little moving parts as possible.

Versioning

Bihourly has 5 snapshots, and daily has 3 snapshots. That means the oldest backup it keeps is 3 days. Any backups past 3 days old will be deleted.

Installation

  1. Unzip
  2. Double click on acorn_install.bat

Changing Settings

This program was made to be immutable. That means from the time you install it, there is no changing of settings. If you want to watch a new folder or change anything, run acorn_install.bat again which will start anew.
