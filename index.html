Acorn

Robust, immutable backup system for your working files. Build on restic.

Simple

Install, setting folders to be watched. In those folders, just add "WORKING" to the filename. Those files will be backed up, every 30 minutes and once daily.

Robust

This program essentially sets up two Windows Task Scheduler that runs generated backup script every 30 minutes and daily. And gives you a GUI to run restic restore commands.

On top of that, this is an immutable program. That means, once you set it up, there's no changing things. It's a fixed size wrench. This is so there is as little moving parts as possible.

Versioning

Bihourly has 5 snapshots

Daily has 3 snapshots

Changing Settings

Got a new drive? Got a new folder you want to watch? New keyword? Download the program, press the uninstall button, and reinstall! All old repos will have to be manually deleted by you.
